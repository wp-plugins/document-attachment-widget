Document attachment widget will display any related attachment, which has been attached to a page or post.

== Description ==

Document attachment is a sidebar widget, which will display a filtered, sorted and ordered list of all related items linked to current post or page. 
This sidebar widget is not just limited to the sidebar of your site. By using the and added the [document-list-attachments] shortcode to a post or page
the widget will then display the list of linked attachments within the documents content area. 

The widget options are:
1) Select attachment file types, which are allowed to be displayed:
	- MS Word
	- MS Excel
	- Adobe PDF
	- Images
	- Adobe Flash
	
2) Display the file size:
	- Show file size
	
3) Display the files description text:
	- Show description text
	
After selecting your options and saving the widget settings, the widget will then only display the attachment mime types which have been selected. 


== Installation ==
1. Old-school: upload the `Document Attachment Widget` folder to the `/wp-content/plugins/` directory via FTP, or Add the plugin via the WordPress Plugins menu.
2. Activate the Plugin
3. Add the Widget to the desired sidebar in the WordPress Widgets menu.
4. Configure the `Document Attachment Widget` options.
5. Add a the shortcode to your post like so: `[document-list-attachments]`
6. Test that the this plugin meets your demanding needs.
7. Rate the plugin and verify that it works at wordpress.org.
8. Leave a comment regarding bugs, feature request at: http://wordpress.org/tags/document-attachment-widget
